## Testing

When you document test prompts, keep them copyable with zero tweaks.

```tsx filename="tests/font-system.test.tsx"
import { expect, test } from "vitest";
import { fontMetrics } from "@/lib/typography";

test("Bangla baseline stays aligned", () => {
  const baseline = fontMetrics.bn.baselinePx;
  expect(baseline).toBeLessThanOrEqual(8);
});
```
